# Dungeon Mayhem (JavaFX)

Фан/учебная desktop-игра в жанре карточного дуэльного “баттлера” в духе Dungeon Mayhem: выбираются персонажи со статами/множителями, игроки по очереди разыгрывают карты (атака/защита/лечение + спецкарты).

## Возможности
- Одиночная игра против бота (AI ходит автоматически, с задержками для “анимации” хода)
- Сетевая игра на 2 игроков: встроенные сервер/клиент, обмен ходами и чат
- Карты: ATTACK / DEFEND / HEAL и набор спецкарт (DOUBLE_ATTACK, SUPER_SHIELD, ULTIMATE_HEAL и др.)
- Персонажи: Варвар / Паладин / Плут / Маг — разные множители атаки/защиты/лечения и “способность” в описании

## Стек
- Java + Maven
- JavaFX (controls, fxml)
- Lombok

## Быстрый старт

### Требования
- JDK 11+ (лучше 17)
- Maven Wrapper в репозитории (mvnw / mvnw.cmd)

### Запуск
Linux/macOS:
```bash
./mvnw javafx:run
```

Windows:
```
mvnw.cmd javafx:run
```
